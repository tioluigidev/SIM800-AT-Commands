# Basic Commands
### Test Command
```
AT
```
Return:
```
OK
```

---
### Set Echo Mode
```
ATE<mode>
```
Return:
```
OK
```
This setting determines whethever or not the modem echoes characters received from serial port.<br/>
- ATE0 - Set echo off<br/>
- ATE1 - Set echo on
